# Installation and Setup (Internal Server Version)

## 1. Downloading the plugin

Download the latest version of the plugin from the **[Releases page](https://github.com/heitorcordeiromaciel/essentialsmultiplayersolution/releases/tag/v2.0.0)**.

## 2. Adding it to your project (client)

Extract the downloaded `.zip` file, open the `Internal Server Version` folder and move the following into your game project:

```Voltseon's Multiplayer Solution/ → [YourGame]/Plugins/```

Your final project structure should look like this:

```
YourGame/
├── Game.exe
├── Data/
├── Plugins/
│ ├── OtherPlugins/
│ └── Voltseon's Multiplayer Solution/ # located inside the client folder
```

No setup required other than installing the plugin, it is completely Plug'N'Play

## Usage

Open the game, open the Menu, select Link Play, either Host a Game or Join an existing Game.
