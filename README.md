# heitorcmacielmultiplayersolution
## Fork of [Voltseon's Multiplayer Solution](https://eeveeexpo.com/resources/1453/) for Pokémon Essentials V21.1

Since the original plugin is no longer maintained, this fork tries to keep it up to date with current Essentials features and plugins, fix bugs and security

## Feature progress: 

- [X] Z-Move Support (DBK Z-Power)
- [X] Dynamax Support (DBK Dynamax)
- [X] Tera Support (DBK Terastallization)
- [ ] Following Pokémon support
- [ ] Overworld Encounters support
- [ ] More battle formats
- [ ] Server security fixes
- [ ] general bugfixes
